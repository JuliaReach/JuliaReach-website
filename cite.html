Citing packages of JuliaReach

Publications

If you find the packages of JuliaReach useful and you employed some of them for work that has been published in peer-reviewed journals, we would like you to acknowledge the use of the respective packages. Each core package has an associated publication that is citable. You can find BibTeX entries in the home page of each package's documentation.

Star us on GitHub

We would also appreciate it, if you starred our repositories on GitHub and helped us spread the word about JuliaReach.
